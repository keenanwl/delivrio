Add hypothesis test

Hypothesis testing (including A/B) testing is a type of statistical analysis to inform decision makers about the impact of choosing one potential option over another.

In a e-commerce and more specificaly, a shipping context, DELIVRIO provides tools to test conversion rates, by tweaking various properties of displayed delivery options.

Examples:

* Test impact of different delivery option ordering

* Test impact of pricing or free shipping

* Test impact of delivery window

* Before fully dropping a carrier, test the impact of the reduced delivery options
